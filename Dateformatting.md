valid Format strings are, every combination of the following letters and signs "MdyHhmst_.-" and whitespace.
please note that this is case sensitive!!!!_

  * y yy yyy yyyy // "8 08 008 2008"   year
  * M MM MMM MMMM // "3 03 Mar March"  month
  * d dd ddd dddd // "9 09 Sun Sunday" day
  * h hh H HH 	  // "4 04 16 16"      hour 12/24
  * m mm		  // "5 05"            minute
  * s ss		  // "7 07"            second
  * t tt		  // "P PM"            A.M. or P.M.
